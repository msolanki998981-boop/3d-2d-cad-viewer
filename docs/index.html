About the App

Precision CAD is built to open and inspect engineering files directly on Android devices. It supports full file import and preview for a broad range of CAD, 3D, and laser-cutting formats.

Android 13 (API 35)

Key Features

  • View 3D models: STL, OBJ, GLB, GLTF, PLY.
  • Inspect 2D engineering drawings: DXF, SVG.
  • Open files from other apps using content and file URIs.
  • Supports local storage access and safe file preview.
  • Hardware-accelerated 3D rendering with OpenGL ES 2.0.

Permissions Used

The app only requests permissions needed to open files and render supported CAD assets.

  • READ_EXTERNAL_STORAGE
  • MANAGE_EXTERNAL_STORAGE
  • READ_MEDIA_IMAGES
  • READ_MEDIA_VIDEO
  • READ_MEDIA_AUDIO
  • INTERNET

Privacy & Security

Privacy is a core part of how Precision CAD works. The app does not collect personal user data, does not require account signup, and does not send file contents to any server by default.

For full coverage of data handling and user rights, read the Privacy Policy.
