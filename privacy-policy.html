Introduction

Precision CAD is an Android application for viewing and inspecting engineering files locally. This policy explains what information the app accesses, why it needs permissions, and how user privacy is protected.

What We Collect

Precision CAD does not collect or store personal information. The app is designed to work with user-selected files on the device and only uses permissions that are necessary for local file access and rendering.

  • File URI access for supported CAD assets
  • Metadata required to render models and drawings
  • No user accounts, login credentials, or analytics are collected

Permissions

The app requests the following permissions so it can open and render files chosen by the user:

  • READ_EXTERNAL_STORAGE: Read files from device storage when user grants access.
  • MANAGE_EXTERNAL_STORAGE: Manage file access for supported CAD assets when allowed.
  • READ_MEDIA_IMAGES, READ_MEDIA_VIDEO, READ_MEDIA_AUDIO: Enable media file access on Android 13+ devices when those assets are opened.
  • INTERNET: Required only for potential external content access and safe WebView support; no personal data is transmitted by default.

How File Data Is Used

Files are opened locally on the device and rendered in the application. Precision CAD does not upload file contents to external servers or use files for any purpose other than display and inspection.

Any temporary rendering or file parsing is done in-app. User files remain under the user's control.

Security

We follow standard Android security practices. The app uses hardware-accelerated rendering and local storage access only when explicitly authorized by the user.

Users should keep their device secure and only open files from trusted sources.

Children

Precision CAD is not directed at children under the age of 13 and does not intentionally collect personal information from minors.

Contact

If you have questions about this policy or need additional information, please use the contact information in the repository or app store listing for Precision CAD.
